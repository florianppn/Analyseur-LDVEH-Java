FICHIERS :

- Avant d'utiliser ant, veuillez vous mettre à la racine du projet.

- Pour compiler le projet : ant compile
- Pour créer un .jar du projet : ant create-jar
- Pour créer un javadoc du projet : ant create-javadoc

- Le .jar du projet se trouve dans le dossier dist/
↳ utiliser ./run.sh via le terminal pour lancer le projet.
- La javadoc du projet se trouve dans le dossier doc/
